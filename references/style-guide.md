# Style Guide: Ancient Chinese History Writing

Supplementary reference for the ancient-chinese-history-writer skill. Contains detailed banned-phrase lists, full sample output, and elaboration on key rules.

## Banned AI Phrases — Complete List

The following phrases and their variants must be avoided in ALL generated articles. They signal "AI-generated" to readers and make prose feel generic and impersonal.

### Abstract significance claims
- 具有重要意义 / 具有深远的历史意义 / 意义重大
- 影响深远 / 产生了深远的影响
- 不可忽视 / 不容忽视
- 值得关注 / 值得深思 / 值得我们反思

### Template openings
- 某某事件是中国历史上著名的……
- 在中国古代历史长河中……
- 众所周知 / 众所周知的是……
- 某某朝代是中国历史上一个重要的朝代……

### Vague transitions
- 从某种程度上说 / 从某种意义上说
- 不可否认的是……
- 毋庸置疑……
- 总的来说 / 综上所述

### Grandiose metaphors
- 历史的车轮 / 历史的潮流
- 以史为鉴 / 以史为镜
- 历史的必然 / 历史的巧合
- 时代的选择 / 历史的考验

### Empty closings
- 值得我们深思
- 给我们留下了深刻的启示
- 至今仍具有重要的现实意义
- 永远铭刻在历史的记忆中

## Preferred Concrete Expressions

| Instead of... | Use... |
|---|---|
| 具有重要意义 | 这件事直接改变了唐朝的权力结构 |
| 影响深远 | 此后一百多年里，藩镇问题始终未能解决 |
| 不可忽视 | 直接导致了朝廷对河北的失控 |
| 众所周知 | （直接陈述事实，不加前缀） |
| 在历史长河中 | （删除，直接写事件） |
| 值得我们深思 | 这个选择最终决定了帝国的走向 |

## Sample Full Article Output

The following illustrates the target style — scene-driven opening, tight background, turning-point narrative, specific impact, and a judgment closing.

---

# 《玄武门之变：一场被逼到极处的权力决战》

武德九年六月四日清晨，长安城北的玄武门外，李世民已经没有多少退路。太子李建成和齐王李元吉即将入宫，如果这一天仍按旧秩序过去，秦王府很可能会被彻底清算。后来的史书常把玄武门之变写成唐太宗登基前的关键一击，但站在当时看，它更像是一场被逼到极处的权力决战：胜者入主东宫，败者连活下去的机会都很小。

唐朝开国才九年，但李渊的三个儿子已经走到了你死我活的地步。问题不在于某个人品性特别恶劣，而在于唐朝建立过程中的权力分配本身就有隐患。李渊以关陇贵族起兵，长子李建成作为太子留守后方，次子李世民在前线打硬仗。太原起兵后，李世民先后平定薛举、刘武周、王世充、窦建德等势力，身边聚拢了房玄龄、杜如晦、尉迟敬德、秦叔宝等一批能臣猛将。军功和谋士集团让秦王府形成了一个事实上的第二权力中心，这已经不是太子和李世民两个人之间的较量，而是两个完整班底之间的结构性冲突。

更关键的是，李渊作为皇帝和父亲，既没有压住太子集团的排挤，也没有控制住秦王集团的膨胀。他试图用"和稀泥"来维持平衡，但唐朝初年的政治环境已经不允许这种模糊状态持续下去。

武德九年六月初四这天，李世民带着尉迟敬德等人在玄武门设伏。李建成和李元吉入宫途经玄武门时察觉异常，但已来不及脱身。李世民一箭射杀李建成，尉迟敬德斩杀了李元吉。随后尉迟敬德全副武装进入宫中，向正在泛舟海池的李渊报告了这一事实。史书记载，李渊当时"大惊"，但很快明白局势已不可逆转。三天后，李世民被立为太子。两个月后，李渊退位为太上皇，李世民登基，是为唐太宗。

整个过程不到一天，但这一天改变的东西，比唐朝此后很多年的战争都多。

玄武门之变直接的结果是李世民掌权，但更深层的改变在于：它摧毁了李渊时期试图维持的权力平衡格局，也让唐朝的皇位继承从此带上了暴力的阴影。李世民之后，唐朝的皇位传承反复出现政变、逼宫、废立，太子之位也成为朝廷最危险的位置之一。这不是李世民的人品问题，而是玄武门开了头之后，后来的皇子们都看清了一条规则：在权力面前，嫡长子继承制随时可以被武力推翻。

另一方面，李世民本人因为背负着"杀兄逼父"的道德包袱，在位期间极度努力地证明自己是天命所归的君主。贞观之治的出现，在一定程度上正是这种压力的结果。他不是第一个有能力的皇帝，但可能是最怕被人说成"不合法"的那一个。

玄武门之变真正值得注意的，不只是李世民的胜利，而是唐初权力分配已经走到无法调和的地步。李渊试图用父亲的身份管理政治冲突，但唐朝开国时的军功结构决定了，这场冲突早晚要用刀剑来解决。李世民不过是被局势推到玄武门的那个人。

## Title Templates — Detailed

For reference when generating a title. Mix and match the structure "Event Name: judgment/subtitle."

### Subtitle patterns by event type

**Succession / coup events (政变类):**
- 《事件名：一场被逼到极处的权力决战》
- 《事件名：胜负只在几个时辰之间》
- 《事件名：当继承规则不再管用》

**War / rebellion events (战争/叛乱类):**
- 《事件名：盛唐为何突然崩裂》
- 《事件名：一场本可避免的溃败》
- 《事件名：表面是叛乱，背后是失衡》

**Diplomacy / treaty events (外交/盟约类):**
- 《事件名：宋朝的屈辱，还是现实的止损？》
- 《事件名：一纸盟约背后的实力账本》
- 《事件名：和平的代价是什么》

**Transition / founding events (改朝换代类):**
- 《事件名：黄袍背后的权力默契》
- 《事件名：一次被制造出来的"天命"》
- 《事件名：旧秩序的终点，新秩序的起点》

## Tone Calibration by Reader

### For general readers (普通读者)
- Avoid obscure historical terminology without brief context
- Explain official titles and institutional terms naturally in the flow
- Use modern language for clarity, but keep the historical atmosphere

### For high school students (高中生读者)
- Keep sentence structure clear but not simplistic
- Provide slightly more background on institutional knowledge (e.g., what "节度使" means in practice)
- Emphasize cause-and-effect logic

### For history enthusiasts (历史爱好者)
- Can use more specialized terminology without explanation
- Can engage with historiographical controversies
- Can mention specific primary sources (《旧唐书》《新唐书》《资治通鉴》 etc.)

## Handling Historical Controversies

When sources disagree or a narrative is disputed, use these patterns:

| Situation | Phrasing |
|---|---|
| Standard historical consensus | 史书记载 / 据《资治通鉴》记载 |
| Disputed detail | 后世对此有不同的解释 / 这一细节的说法并不统一 |
| Unofficial history / anecdote | 据野史说法 / 后世笔记中另有记载 |
| Modern scholarly disagreement | 学界对此仍有争议 |
| Certain core fact amid uncertainty | 可以确定的是…… |
| Speculative but plausible | 从当时局势看，这很可能意味着…… |

Never present a disputed anecdote (especially from novels like 《三国演义》 or TV dramas) as established historical fact.
